\doxysubsection{Detailed Description}
\doxylink{structImage}{Image} loading, preprocessing, and normalisation. 

Uses libpng for PNG loading (no SDL2\+\_\+image required). SDL2 is used only for rotation rendering.

Pipeline applied before \doxylink{structCNN}{CNN} inference:\+ load PNG → grayscale → binarize → (optional rotate) → resize 28×28 → float\+[784]\+

Convention:\+ black ink = 1.\+0, white background = 0.\+0.
