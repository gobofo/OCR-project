\doxysubsection{Detailed Description}
Binary serialisation and deserialisation of \doxylink{structCNN}{CNN} weights. 

The model file format is a flat binary dump of the \doxylink{structCNNWeights}{CNNWeights} struct, preceded by a 4-\/byte magic number and a 4-\/byte version field:\+

\+[magic:\+ 4 bytes \textquotesingle{}OCRC\textquotesingle{}]\+ \+[version:\+ uint32\+\_\+t]\+ \+[\doxylink{structCNNWeights}{CNNWeights}:\+ sizeof(\+CNNWeights) bytes]\+

This allows fast save/\+load with a single fwrite/\+fread call and easy detection of incompatible files.
