\doxysubsection{Detailed Description}
Letter segmentation:\+ detect crossword grid and extract letter cells. 

Pipeline:\+
\begin{DoxyEnumerate}
\item Detect the crossword grid bounding box in the binarised image.
\item Identify grid lines to compute cell size and origin.
\item Return an ordered array of \doxylink{structBoundingBox}{Bounding\+Box}, one per cell, left-\/to-\/right, top-\/to-\/bottom.
\item Each bounding box can then be cropped, resized, and fed to the \doxylink{structCNN}{CNN}.
\end{DoxyEnumerate}

Alternatively, for images with clearly separated letter blobs, the connected-\/component analysis path extracts letter regions directly.
