\doxysubsection{Detailed Description}
Training dataset loader. 

Expects a root directory with one sub-\/directory per letter:\+

training\+\_\+data/\+ A/\+ image1.\+png image2.\+png … B/\+ … … Z/\+ …

Each image is loaded with SDL2, converted to grayscale, and resized to CNN\+\_\+\+IMG\+\_\+H × CNN\+\_\+\+IMG\+\_\+W. The resulting pixel values are normalised to \+[0, 1]\+.

Parallelism:\+ loading can be parallelised over POSIX threads; the number of worker threads is controlled by the caller.

Load all images from {\ttfamily root\+\_\+dir} into a \doxylink{structDataset}{Dataset}. 

Opens sub-\/directories named A–Z. For each PNG/\+\+JPG image found, loads it with SDL2, converts to grayscale, resizes to 28×28, normalises to \+[0, 1]\+, and appends a \doxylink{structSample}{Sample} to the dataset. Non-\/image files are silently skipped.
