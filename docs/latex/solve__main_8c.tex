\doxysubsection{Detailed Description}
Entry point for the OCR + crossword-\/solver binary. 

Usage:\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{./solve\ -\/-\/image\ <path>\ [-\/-\/model\ <path>]\ [-\/-\/words\ <word1,word2,...>]\ [-\/v]}

\end{DoxyCode}


Options:\+ --image $<$path$>$ (required) Input crossword image. --model $<$path$>$ (optional) Model file to load. If absent, picks the most recent .bin in models/\+. --words 
\begin{DoxyItemize}
\end{DoxyItemize}list of words to search for. --verbose /\+ -\/v Print per-\/cell recognition details and grid.

Pipeline:\+
\begin{DoxyEnumerate}
\item Load model.
\item Load and preprocess the image (grayscale, binarize).
\item Segment letter cells.
\item Recognise each cell with the \doxylink{structCNN}{CNN}.
\item Build a character grid.
\item Solve the word search (if --words is given).
\end{DoxyEnumerate}

Exit codes:\+ 0 Success. 1 Argument error. 2 Model error (not found /\+ cannot load). 3 \doxylink{structImage}{Image} load error. 4 Segmentation error.
