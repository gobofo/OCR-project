Load all images from {\ttfamily root\+\_\+dir} into a \doxylink{structDataset}{Dataset}. 

Opens sub-\/directories named A–Z. For each PNG/\+\+JPG image found, loads it with SDL2, converts to grayscale, resizes to 28×28, normalises to \+[0, 1]\+, and appends a \doxylink{structSample}{Sample} to the dataset. Non-\/image files are silently skipped.
