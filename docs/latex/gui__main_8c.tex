\doxysubsection{Detailed Description}
SDL2 graphical interface for the OCR crossword solver. 

Provides a 1280×800 window with three text-\/input rows:\+
\begin{DoxyItemize}
\item {\bfseries{\doxylink{structImage}{Image}}} — path to the PNG crossword image
\item {\bfseries{Modèle}} — path to the trained \doxylink{structCNN}{CNN} model (.bin)
\item {\bfseries{Mots}} — comma-\/separated list of words to find
\end{DoxyItemize}

Clicking {\bfseries{Charger}} (or pressing Enter) loads the file. Clicking {\bfseries{Chercher}} (or pressing Enter in the words field) runs the full OCR pipeline and overlays red rectangles on each found-\/word cell. The original image pixels are never modified.

\begin{DoxyParagraph}{Keyboard shortcuts}

\begin{DoxyItemize}
\item {\bfseries{Tab}} — cycle focus between fields
\item {\bfseries{Ctrl+V}} — paste from clipboard into the focused field
\item {\bfseries{Enter}} — validate /\+ trigger action for the focused field
\item {\bfseries{Escape}} — clear focus
\item {\bfseries{Ctrl+Q}} — quit
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Dependencies}
SDL2, SDL2\+\_\+ttf, libpng — plus the project\textquotesingle{}s own \doxylink{structCNN}{CNN} /\+ segment /\+ solver.
\end{DoxyParagraph}

Predict the letter in a grid cell using Test-\/\+Time Augmentation (TTA). 

Runs TTA\+\_\+N forward passes centred on the bounding-\/box centre, each with a small ±2 px spatial shift, averages the softmax outputs, and returns the argmax class index (0=\textquotesingle{}A\textquotesingle{} … 25=\textquotesingle{}Z\textquotesingle{}).

The crop window is {\ttfamily cell\+\_\+size} × {\ttfamily cell\+\_\+size} (grid pitch) so that every letter sees a consistent white border regardless of how tight the connected-\/component bounding box is. If {\ttfamily cell\+\_\+size} is 0, a 35\%-\/padding heuristic is used instead.
